# GPT Helper

一个用于 ChatGPT 网页版的轻量级浏览器扩展。它会在对话页面右侧添加一个悬浮目录，按用户提问生成导航点，方便在长对话中快速跳转和定位上下文。

## 功能特性

- 自动识别 ChatGPT 对话中的用户提问。
- 在页面右侧显示紧凑的悬浮导航点。
- 鼠标悬停或键盘聚焦时显示提问摘要。
- 点击导航点可跳转到对应提问。
- 当前可见提问会自动高亮。
- 支持较长对话中的懒加载定位，会先估算滚动位置再对齐目标提问。
- 兼容 ChatGPT 浅色、深色主题。

## 支持站点

扩展会在以下页面注入内容脚本：

- `https://chatgpt.com/*`
- `https://chat.openai.com/*`

## 项目结构

```text
.
├── manifest.json   # Chrome 扩展清单，声明注入站点和脚本
├── content.js      # 内容脚本，负责收集提问、渲染目录和滚动定位
└── content.css     # 悬浮目录的样式
```

## 安装和使用

1. 打开 Chrome 或其他 Chromium 内核浏览器。
2. 进入 `chrome://extensions/`。
3. 开启右上角的「开发者模式」。
4. 点击「加载已解压的扩展程序」。
5. 选择本项目目录。
6. 打开或刷新 ChatGPT 对话页面。

安装后，进入 ChatGPT 对话页即可在页面右侧看到一列导航点。将鼠标移到导航点上可以查看对应提问摘要，点击后跳转到该提问。

## 开发说明

本项目不依赖构建工具，修改文件后在扩展管理页点击刷新扩展，再刷新 ChatGPT 页面即可验证效果。

主要配置位于 `content.js` 顶部的 `CONFIG`：

- `MAX_TITLE_LENGTH`：导航提示中展示的提问摘要长度。
- `TURN_SELECTOR`：ChatGPT 对话轮次节点选择器。
- `REBUILD_DELAY_MS`：DOM 变化后重建目录的防抖时间。
- `PENDING_SCROLL_ATTEMPTS`：目标提问尚未加载时的滚动重试次数。
- `SCROLL_TARGET_TOLERANCE_PX`：滚动对齐的容忍像素值。

样式可在 `content.css` 中调整，包括导航位置、提示卡片、亮暗色主题和移动端宽度。

## 工作原理

扩展通过内容脚本扫描 ChatGPT 页面中的 `conversation-turn-*` 节点，筛选出用户消息并为每个提问生成一个导航项。页面内容变化时，`MutationObserver` 会触发目录重建；滚动过程中，`IntersectionObserver` 会更新当前高亮项。

对于尚未加载到 DOM 中的历史提问，扩展会根据提问序号估算滚动位置，等待页面加载对应节点后再进行精确对齐。

## 注意事项

- ChatGPT 页面 DOM 结构更新后，选择器可能需要同步调整。
- 本扩展只在浏览器本地运行，不会发送或存储对话内容。
- 当前版本没有后台脚本、弹窗页面或配置界面。
